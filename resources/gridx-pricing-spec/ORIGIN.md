# GridX Pricing API Specification

These files were derived from the GridX developer documentation and originally maintained in the `gridx-api-specs` repository (https://github.com/grid-coordination/gridx-api-specs).

They are embedded here so that `git clone` + `clojure -M:nrepl` works without needing a second repository checkout.

## Contents

- `openapi.yaml` — OpenAPI 3.0 specification for the GridX Pricing API
- `schemas/` — JSON Schema definitions for pricing data types
- `examples/` — Sample API responses
